# 3_Proyecto_Riesgo_Relativo
En el actual escenario financiero, la alta demanda de crédito en el banco "Super Caja" ha abrumado al equipo de análisis de crédito. En este proyecto Optimice la eficiencia y precisión del riesgo crediticio mediante el análisis de datos. Facilité decisiones informadas en la concesión de crédito, mejorando la efectividad del proceso.
